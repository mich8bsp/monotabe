Monotabe - Guitar playing helper

Native application (Linux build with possible android/mac/windows in the future) written in high performance language.

Basic functionality:

Show a library of songs divided by guitar and bass.
Each song can contain media such as youtube video link or embedding, spotify link or embedding, tabs in PDF format, mp3 audio file. The UI should allow playing the song from either source and display the tabs from PDF file.

Advanced functionality:

Sync between tabs and music. Choosing tabs PDF file and MP3 music file and giving it to LLM to analyze, the LLM should analyze the tab format and map it to song length and provide a map of song cadence which will be used when displaying the PDF to determine when to auto-scroll as the song is playing.
The end goal is to provide a seamless experience of playing along to the song while tabs are shown and auto-scrolled without the need to stop playing guitar to advance the tabs.
